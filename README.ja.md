# 音声入力 (VoiceInput) for Mac

[English](README.md) · [繁體中文](README.zh-Hant.md) · **日本語**

**Mac 用のオフライン音声入力** —— ホットキーを押して話すと、テキストがどのアプリでもカーソル位置に直接入力されます。端末内で動く Whisper、**60 以上の言語**に対応、**完全オフライン** —— 音声が Mac から出ることはありません。**中国語・日本語・韓国語**の精度はこの種のツールでは群を抜きます。

このリポジトリは**公開リリース**を配布します。

## ダウンロード

最新の `VoiceInput-x.y.zip` を **[Releases](../../releases)** から入手してください。

## インストール

1. ダウンロードして解凍 →  **VoiceInput.app** を「アプリケーション」フォルダへドラッグ
2. 起動（Apple Developer ID 署名 + 公証済みのため、Gatekeeper の警告は出ません）
3. 初回ガイドに従って、認識モデルのダウンロードと「マイク」「アクセシビリティ」を許可
4. どのアプリでもホットキーを押して話すと、カーソル位置にテキストが挿入されます

## 動作環境

- macOS 14 以降
- 最適な速度のため Apple Silicon（M シリーズ）を推奨
- 初回に認識モデルを一度ダウンロードします（約 574MB）

## 特長

- 端末内 Whisper（large-v3-turbo）、多言語で高精度
- 完全オフライン・プライバシー保護
- どのアプリにも入力 · CJK 組版の最適化（和欧文の空白・句読点・簡繁）
- 単語登録 · 修正学習 · 二言語のワンタップ切替
